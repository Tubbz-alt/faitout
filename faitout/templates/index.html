Clean a connection ------------------ Clean one of your connection by dropping and re-creating the database curl {{ config.URL }}{{ url_for('clean_database', db_name='db_name') }} Database db_name has been cleaned Drop a connection ------------------ Drop one of your connection by dropping the database curl {{ config.URL }}{{ url_for('drop_database', db_name='db_name') }} Database db_name has been dropped Get some stats about the current faitout ---------------------------------------- Check out: {{ config.URL }}{{ url_for('stats') }} Get some stats for your IP -------------------------- Check out: {{ config.URL }}{{ url_for('info') }} Faitout and other tools ======================= faitout and python ------------------ Install the requests module from : http://www.python-requests.org/ You may then set the SQLAlchemy URL in one line as follow: DATABASE_URL = get('{{ config.URL }}{{ url_for('token') }}').text faitout and Travis CI --------------------- Travis CI is a continuous integration service: https://travis-ci.org/ Simply add the following to your .travis.yml env: - DATABASE_URL=$(curl {{ config.URL }}{{ url_for('token') }}) {% endblock %}
